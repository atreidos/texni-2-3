# Калькулятор стоимости разработки (Фрилансер) — V1

Минимальная версия калькулятора стоимости услуг веб-разработки для фрилансера. Чистый HTML, CSS и JavaScript без библиотек.

## Что умеет калькулятор

- Выбор услуги (лендинг, визитка, калькулятор, корпоративный сайт, другое).
- Для «Другое» — поле ввода своей базовой цены.
- Ползунок количества экранов/страниц (1–20) с живым отображением.
- Чекбоксы: Telegram + форма, SEO базовая, уникальный дизайн, срочность +50%.
- Итоговая цена обновляется при любом изменении.
- Кнопка «Получить коммерческое предложение» — текст заявки в `alert` и данные в `console.log`.
- Кнопка «Распечатать расчёт» — открывает окно печати (`window.print()`).

Цены и лимиты экранов заданы в объекте `prices` в начале файла `script.js`.

## Запуск одним кликом (двойной клик по index.html)

1. Открой папку проекта в проводнике Windows (например: `c:\dev\study\unit1-3`).
2. Найди файл **index.html**.
3. Сделай **двойной клик** по **index.html**.
4. Страница откроется в браузере по умолчанию — калькулятор готов к работе.

Никакой установки, сервера или сборки не нужно: всё работает локально в браузере.

## Файлы проекта

- **index.html** — разметка и форма калькулятора.
- **style.css** — стили страницы.
- **script.js** — логика расчёта и обработчики (все цены в одном объекте).
- **README.md** — этот файл с инструкцией.

## Важно

- Файлы `index.html`, `style.css` и `script.js` должны лежать в одной папке.
- Если при двойном клике открывается не браузер, а редактор: щёлкни по файлу правой кнопкой → «Открыть с помощью» → выбери браузер (Chrome, Edge, Firefox и т.д.).
