## Ответы на уточнения по задаче "Сбросить калькулятор"

1. Кнопка `Сбросить калькулятор` размещается рядом с кнопками `Получить коммерческое предложение` и `Распечатать расчёт`.
2. Под "цена становится базовой" понимается пересчет как для стартового набора: без доп. опций и с ползунком на значении `5`.
3. После сброса итоговая цена должна обновляться сразу, без дополнительных действий пользователя.
4. Для фиксации структуры используется файл `ARCHITECTURE.md`.
