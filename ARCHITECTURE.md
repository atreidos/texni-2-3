# Архитектура проекта

Проект представляет собой одностраничный калькулятор стоимости услуг на чистом HTML/CSS/JS без сборки и зависимостей.

## Структура файлов

- `index.html` — разметка интерфейса калькулятора:
  - выбор услуги;
  - поле своей цены для услуги "Другое";
  - ползунок количества экранов;
  - чекбоксы дополнительных опций;
  - блок итоговой цены и детализации;
  - кнопки действий (коммерческое предложение, печать, сброс);
  - модальное окно отправки заявки.
- `style.css` — стили страницы, формы, кнопок, детализации, модального окна, адаптивности и печати.
- `script.js` — вся логика приложения:
  - конфигурация цен в объекте `prices`;
  - вычисление итоговой стоимости (`calcTotal`);
  - отрисовка детализации (`renderBreakdown`);
  - синхронизация UI (`updateDisplay`);
  - обработчики кнопок и событий формы;
  - поведение модального окна.
- `README.md` — краткая инструкция по запуску и возможностям.

## Логика расчёта

1. Пользователь выбирает услугу.
2. Для выбранной услуги берутся базовая цена и лимит экранов из `prices`.
3. Если экранов больше лимита, добавляется доплата за каждый дополнительный экран.
4. Суммируются выбранные опции.
5. При включённой срочности к итогу применяется множитель.
6. Если применён промокод, к итогу применяется скидка (`activeDiscount`).
7. Итог округляется и отображается вместе с детализацией.

## Логика промокодов

- Промокоды хранятся в объекте `promoCodes` (ключ — код, значение — доля скидки: `0.10` = 10%).
- Активных промокодов: `SALE10` (10%), `SUPER20` (20%).
- Переменная `activeDiscount` хранит текущую скидку (0 = нет скидки).
- Скидка применяется последней в `calcTotal` — после срочности.
- После успешного применения поле ввода и кнопка блокируются (чтобы нельзя было ввести второй раз).
- При сбросе калькулятора промокод тоже сбрасывается.

## Логика сброса калькулятора

- Кнопка `Сбросить калькулятор` вызывает подтверждение `Вы уверены?`.
- После подтверждения:
  - все чекбоксы опций снимаются;
  - ползунок экранов возвращается на значение `5`;
  - промокод очищается, скидка обнуляется;
  - интерфейс пересчитывается и цена становится базовой (без доп. опций).
