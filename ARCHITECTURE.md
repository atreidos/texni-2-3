# Архитектура проекта: Калькулятор стоимости разработки (Фрилансер)

Учебный проект — одностраничное приложение на чистом HTML/CSS/JS, без фреймворков и сборщиков.

## Структура файлов

```
unit1-3/
├── index.html   — разметка и структура страницы
├── style.css    — стили
├── script.js    — вся логика
└── README.md    — документация
```

---

## `index.html` — структура страницы

Страница содержит два основных блока:

**1. Форма-калькулятор** (`#calc-form`) внутри `.wrapper`:
- `<select #service>` — выбор услуги (лендинг, визитка, калькулятор, корпоративный сайт, другое)
- `#other-price-wrap` — скрытое поле для ввода своей цены (показывается только при выборе "Другое")
- `<input type="range" #screens>` — ползунок количества экранов (1–20)
- Чекбоксы опций: Telegram, SEO, уникальный дизайн, срочность
- Блок `#total-price` — итоговая цена
- `<ul #price-breakdown>` — детализация расчёта (рендерится динамически)
- Три кнопки: "Получить коммерческое предложение", "Распечатать расчёт", "Сбросить калькулятор"

**2. Модальное окно** (`#modal`) — форма заявки с полями Имя, Телефон, Telegram и кнопкой отправки

## `script.js` — логика (218 строк)

| Часть | Что делает |
|---|---|
| `prices` (объект) | Единое хранилище всех цен, базовых лимитов экранов, стоимостей опций |
| DOM-переменные | Все элементы получаются один раз при загрузке |
| `toggleOtherPrice()` | Показывает/скрывает поле своей цены |
| `getServiceData()` | Возвращает базовую цену и лимит экранов для выбранной услуги |
| `calcTotal()` | Считает итог: база + доп. экраны + опции + срочность |
| `renderBreakdown()` | Рендерит детализацию расчёта списком под итогом |
| `updateDisplay()` | Обновляет ползунок + итог + детализацию одним вызовом |
| `getFormData()` | Собирает все данные формы в объект для заявки |
| `onOfferClick()` | Открывает модальное окно, очищает его поля |
| `closeOfferModal()` | Закрывает модальное окно |
| `onSendRequestClick()` | Валидирует и "отправляет" заявку (пока без бэкенда) |
| `onPrintClick()` | Вызывает `window.print()` |
| `onResetClick()` | Через `confirm("Вы уверены?")` сбрасывает чекбоксы, ставит ползунок на 5 и сразу обновляет итог |
| `init()` | Подписывает все обработчики событий, запускается по `DOMContentLoaded` |

## `style.css` — стили (537 строк)

Организован по секциям:
- **CSS-переменные** в `:root` — цвета, радиусы, тени
- Базовые стили (`body`, `.wrapper`, `h1`)
- Стили формы: `select`, `input`, `range`, `checkbox`
- Блок итога `.total` и детализация `.price-breakdown`
- **Модальное окно** — анимированный overlay с backdrop-filter
- Сообщения об ошибке/успехе в модалке
- `@media (max-width: 480px)` — адаптация под мобильные
- `@media print` — стили для печати

## Общая логика работы

1. При любом изменении в форме → `updateDisplay()` пересчитывает итог и перерисовывает детализацию
2. Кнопка "Получить КП" → открывается модальное окно с формой заявки
3. Модалка закрывается по кнопке ×, по клику на оверлей или после "отправки"
4. Кнопка "Распечатать" → `window.print()` с отдельными CSS-стилями для печати
5. Кнопка "Сбросить калькулятор" → подтверждение через `confirm`, затем сброс всех опций и возврат к базовому состоянию по ползунку (5)

Проект полностью самодостаточен — открывается двойным кликом по `index.html`, никакого сервера не нужно.
